# Hi, I'm Aaron Avila

Software Engineering student focused on full-stack web applications, clean architecture, and practical user interfaces.

I work mainly with Vue, TypeScript, Java, C#, REST APIs, and database-backed applications. My current public work is centered on **FreshGuard ColdTrack**, a platform for refrigerated shipment management, sensor monitoring, alerts, and operational dashboards.

## Tech Stack

**Frontend:** Vue, TypeScript, JavaScript, Vite, PrimeVue, PrimeFlex, HTML, CSS  
**Backend:** Java, Spring Boot, C#, .NET, REST APIs, JWT authentication  
**Data and tools:** MySQL, OpenAPI/Swagger, Git, GitHub, automated tests  
**Practices:** DDD, CQRS, layered architecture, component-based design, i18n, API integration

## Featured Work

### FreshGuard ColdTrack

ColdTrack is an academic open source project for managing refrigerated shipments and monitoring operational data. The platform includes frontend applications, backend services, landing pages, authentication, shipment workflows, sensor telemetry, alerts, dashboards, reports, and multilingual support.
